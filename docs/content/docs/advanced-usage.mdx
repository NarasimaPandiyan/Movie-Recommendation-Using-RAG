### Customizing the Vector Search

If you want to add more metadata fields for filtering, you must update the **Atlas Vector Search Index** definition in the MongoDB Atlas UI.

<Callout type="important">
  Any changes to your metadata fields (e.g., adding `Director` or `Year`) must be indexed before they can be used in your queries.
</Callout>

```json
{
  "fields": [
    {
      "type": "vector",
      "path": "embedding",
      "numDimensions": 3072,
      "similarity": "cosine"
    },
    { "type": "filter", "path": "metadata.rating" },
    { "type": "filter", "path": "metadata.languages" }
  ]
}
```
