# Movie Recommendation AI 🍿

Welcome to the documentation for your **Movie Recommendation RAG** system. This project is a state-of-the-art implementation of semantic retrieval, combining the power of **MongoDB Atlas** for vector search and **Google Gemini 2.5 Flash** for intelligent synthesis.

  <Card title="Scale with MongoDB Atlas" description="A cloud-native vector store capable of handling millions of movie nodes." />
